Velvexa is a full-stack clothing e-commerce web application built using React, Tailwind CSS, Spring Boot, and MySQL. The platform allows users to browse products, register and login securely, and place orders using RESTful APIs.

## Features
- Responsive UI built with React and Tailwind CSS  
- Secure authentication using Spring Security and JWT  
- Product listing and order management  
- RESTful backend APIs using Spring Boot  
- MySQL database integration  

## Tech Stack
Frontend: React, Tailwind CSS  
Backend: Spring Boot, Spring Security, JWT  
Database: MySQL  

## Modules
- User Registration & Login  
- Product Catalog  
- Cart & Order Processing  
- Role-based Authentication  

## How to Run

### Backend
1. Import project into Spring Tool Suite / IntelliJ  
2. Configure MySQL in `application.properties`  
3. Run the Spring Boot application  

### Frontend
1. Navigate to frontend folder  
2. Run `npm install`  
3. Run `npm start`
